# BillForge

Invoice & Proposal tool built for Indian freelancers.

**Live at:** [billforge](https://github.com/varshithhegde/billforge)

## Features

- **Invoice + Proposal mode** — switch between document types in one click
- **Live A4 preview** — see changes reflected instantly  
- **4 templates** — Minimal, Classic, Bold, Elegant
- **GST ready** — CGST+SGST or IGST with 0/5/12/18/28% rates
- **Bank + UPI details** — printed directly on the invoice
- **PDF export** — pixel-perfect A4 download, no watermarks
- **Auto-save** — data persists in localStorage

## Stack

- Next.js 15 (App Router)
- Tailwind CSS v4 + shadcn/ui
- Zustand (state + persistence)
- jsPDF + html2canvas (PDF export)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and click **Open Builder**.
